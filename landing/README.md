# MyCargoLens — Marketing Site

MyCargoLens marketing site built with Next.js 15 (App Router), TypeScript, Tailwind CSS v4, and Framer Motion.

## Dev

```bash
npm install
npm run dev
```

Opens at http://localhost:3100

## Build

```bash
npm run build
npm start
```

## Docker

```bash
docker build -t mycargolens-landing .
docker run -p 3000:3000 mycargolens-landing
```
